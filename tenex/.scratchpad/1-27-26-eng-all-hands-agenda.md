## 4. Standards Pipeline Overview

High-level discussion on how things become standards.

```
┌─────────────┐     ┌─────────────────┐     ┌─────────────┐
│ Experiment  │────▶│   Recommended   │────▶│  Standard   │
│             │     │ (team member    │     │             │
│             │     │  sponsors it)   │     │             │
└─────────────┘     └─────────────────┘     └─────────────┘
```

**Key points:**
- Experiments: Try it, share learnings
- Recommended: Someone on the team vouches for it
- Standard: Adopted across all projects
