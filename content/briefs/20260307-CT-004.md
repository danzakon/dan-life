## Core Insight
Autonomous coding has a hidden cost: entropy. More autonomy means more decisions you didn't anticipate, more codebase drift, more style inconsistencies. The agents that write code aren't the ones who maintain it. Managing entropy — through consistent context (CLAUDE.md), checkpoints (Plan mode), smaller focused agents, and human review — is the unglamorous skill that separates codebases that survive from ones that rot.
